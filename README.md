# react-moment2-frontend

Detta är ett litet React + TypeScript-projekt (byggt med Vite) som visar en enkel todo-app. Syftet är att demonstrera komponentstruktur, state-hantering och en enkel service för todos.

- Kör projektet med `npm install` och `npm run dev`.
- Källkod finns i `src/` med komponenter under `src/components` och tjänster under `src/services`.
